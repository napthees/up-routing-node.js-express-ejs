# Description of the application "Up routing with node.js, express, bootstrap & ejs!"

Website Pages Routing with Node.js, Express, Bootstrap and EJS

## Dependencies
```
  body-parser@1.18.3
  bootstrap@4.1.3
  ejs@2.6.1
  express@4.16.3
  jquery@3.3.1
  popper.js@1.14.4
```
